# 贡献者行为准则

## 我们的承诺

为营造开放友好的社区，作为贡献者和维护者，我们承诺：参与本项目的每个人都不受骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份。

我们承诺以促进开放、包容、多元化、健康社区的方式行事和互动。

## 我们的准则

有助于营造正面环境的行为示例：

- 对他人展现同理心与善意
- 尊重不同意见、观点与经验
- 给予建设性反馈，也优雅地接受反馈
- 承担责任，向受我们错误影响者道歉，并从中学习
- 关注的不仅是个人，更是整个社区的整体利益

不可接受的行为示例：

- 性化语言或 imagery、性关注或挑逗
- 钓鱼、侮辱或贬损评论，人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人私人信息
- 其他可被合理认定为不适合专业环境的行为

## 执行责任

社区维护者有权澄清并执行本准则，可酌情删除、编辑或拒绝不符合本准则的评论、提交、代码、wiki 编辑、Issue 和贡献。

## 适用范围

本准则适用于所有项目空间，并在个人在公共空间代表项目或社区时同样适用。

## 执行

辱骂、骚扰或其他不可接受行为可通过私信联系项目维护者举报。所有投诉都将被及时公正地审查处理。

维护者有义务尊重举报者隐私与安全。

## 来源

本行为准则改编自 Contributor Covenant 2.1，中文翻译仅供参考。
