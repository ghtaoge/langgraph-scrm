# 客户管理功能

SCRM 客户管理模块帮助企业构建完整的客户画像，实现精细化运营。

## 客户画像
- 基础信息：姓名、手机、来源渠道、添加时间
- 行为标签：浏览记录、咨询记录、购买记录自动生成标签
- 自定义字段：企业可扩展自定义属性（如行业、规模、职位）

## 客户分层
- RFM 模型：按最近消费时间、消费频率、消费金额自动分层
- 阶段管理：线索 → 意向 → 谈判 → 成交 → 复购，全生命周期管理
- 优先级推荐：AI 根据客户活跃度和意向度推荐优先跟进对象

## 客户跟进
- 跟进任务：系统自动生成跟进提醒，避免遗漏
- 跟进记录：每次沟通内容、客户反馈结构化记录
- 协同跟进：多人协作跟进同一客户，信息共享

## 客户分配
- 公海池：未跟进客户自动回收公海，重新分配
- 分配规则：按区域、行业、轮询等规则自动分配销售
- 离职继承：员工离职客户一键转交，保留全部跟进记录
