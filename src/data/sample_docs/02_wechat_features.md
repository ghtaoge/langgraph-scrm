# 微信功能说明

SCRM 系统支持丰富的微信生态功能，覆盖客户触达、营销、服务全场景。

## 企业微信客户管理
- 外部联系人：员工通过企业微信添加客户，客户信息自动同步到 SCRM
- 客户标签：支持手动打标和规则自动打标（如"高意向""已成交"）
- 客户群：支持创建客户群，群成员管理、群发通知

## 朋友圈
- 企业朋友圈：管理员统一发布内容，员工朋友圈同步展示
- 内容素材库：图文、视频、链接素材集中管理
- 互动追踪：客户点赞、评论、点击行为自动记录

## 社群运营
- 社群 SOP：按阶段（新群、活跃期、衰退期）配置运营动作
- 自动回复：入群欢迎语、关键词自动应答
- 群活码：一个活码对应多个群，满员自动建新群

## 会话存档（风控）
- 会话存档：员工与客户聊天记录合规存档（需客户同意）
- 敏感词识别：自动识别价格泄露、辱骂、泄密等风险消息
- 违规预警：高风险消息实时上报主管审核
