# SCRM 产品功能概述

SCRM（社交客户关系管理）系统是企业微信生态下的客户运营平台，核心能力包括：

## 客户管理
- 客户画像：自动聚合客户基础信息、标签、互动记录、消费行为
- 客户分层：支持按 RFM 模型、自定义标签对客户分层运营
- 客户继承：员工离职后客户可一键转交，避免客户流失

## 微信生态集成
- 企业微信客户连接：添加客户为外部联系人，支持单聊、群聊
- 朋友圈营销：企业朋友圈统一发布、客户触达数据回传
- 社群运营：社群 SOP、自动欢迎语、关键词自动回复

## 营销自动化
- 任务引擎：基于客户行为触发营销任务（如浏览后未下单推送优惠券）
- 话术库：标准化销售话术，新员工快速上手
- 线索评级：AI 多轮对话评估线索质量，自动评分分级

## 数据分析
- 客户漏斗：从触达到成交的全链路转化分析
- 员工效能：回复及时率、客户跟进数、成交转化等指标
- 风控审计：敏感词识别、会话存档、违规预警
