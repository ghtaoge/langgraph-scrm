# 订单追踪说明

SCRM 订单模块提供从下单到签收的全链路追踪能力。

## 订单状态
- created：已创建（待支付）
- paid：已支付（待发货）
- shipping：已发货（运输中）
- delivered：已签收
- closed：已关闭（完成或取消）
- exception：异常（需人工介入）

## 物流追踪
- 实时查询：对接主流物流公司 API，物流信息实时同步
- 节点提醒：关键节点（发货、到达、派送、签收）自动通知客户
- 异常告警：物流停滞超 48 小时、派送失败等异常自动告警销售跟进

## 订单问题处理
- 发货延迟：超承诺时效未发货，系统自动升级并通知主管
- 物流异常：运输破损、丢失，可发起补发或退款流程
- 规格不符：实物与描述不符，支持 7 天换货

## 支付问题
- 支付失败：自动重试 + 切换支付渠道，失败订单保留 30 分钟
- 重复支付：系统自动识别并原路退回多扣款项
- 发票管理：支持电子发票自动开具，可按订单或合并开票
